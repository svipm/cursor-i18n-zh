软件声明

  • 本项目是第三方开源本地化工具, 不是 Cursor 或 Anthropic 官方产品, 也未获得其授权或背书.
  • 汉化会修改本机应用资源文件. 安装前必须创建当前软件版本的完整备份, 并通过文件大小和 SHA256 校验.
  • 安装、备份和恢复可能自动结束 Cursor 或 Claude Desktop 进程. 操作前必须保存尚未完成的工作.
  • 目标软件升级后可能需要重新汉化. 因上游资源变化造成的显示异常或兼容性风险由使用者自行承担.

隐私说明

  • 工作台不包含遥测、行为分析或广告组件, 不上传个人文件、备份内容和操作日志.
  • 软件检测、备份、校验和恢复均在本机执行. 备份只保存在界面显示的本地目录.
  • Cursor 用量监控只读本机登录状态. 令牌仅在 Rust 后端内存中使用, 只发送给 Cursor 自有接口, 不返回前端、不写日志、不落盘.
  • 扩展管理只读取用户选择范围内的 MCP 与 Skill. MCP 密钥和请求头不返回前端, 配置修改前会创建本地备份.
  • 版本检查只向 GitHub 请求公开发行版信息, 不携带 Cursor 或 Claude 账号信息, 不自动下载、不静默安装、不强制更新.
  • GitHub 头像仅在打开关于页时按需加载. 不同意本说明可以直接退出, 工作台不会继续初始化.
